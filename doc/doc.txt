# Adresses2Pickles
# Index les fichiers adresses-*.csv
# Utilisation python adresses2pickles.py
# Les adresses doivent être dans le sous répertoire data/adresses
# Le répertoire peut être changé dans config.py adresse_path
# Création des fichiers *.pickle dans le même répertoire

# Cedex2Pickles
# Index le fichier liste-des-cedex.csv
# Utilisation : python cedex2pickle.py data/cedex/liste-des-cedex.csv
# Le répertoire peut être changé dans config.py cedex_path
# Création du fichier *.pickle dans le même répertoire
# Aide : python cedex2pickle.py --help

# AdressesMatcher
# Couple le fichier ps-tarifs*.csv et adresses-*.pickle
# Utilisation : python adressesmatcher.py data/ps/ps-tarifs-21-03.csv
# Génére le fichier data/ps/ps-tarifs-21-03-adresses.csv complété avec les adresses pour tous les départements métropolitains + Corse
# -c utilise le ficher de cache ps-adresses.csv ainsi que le geocoding
# -d indique la liste des départements à matcher
    # La liste doit être au format des listes Python sans espace
    # Pour la Corse 2A = 201 et 2B = 202
# Exemple pour le département 01 python adressesmatcher.py data/ps/ps-tarifs-21-03.csv -d [1]
    # Attention le département ne doit pas commencer par 0, [1] != [01]
# Exemple pour une liste de département python adressesmatcher.py data/ps/ps-tarifs-21-03.csv -d [1,38,48]
# Exemple pour une liste de département consécutifs python adressesmatcher.py data/ps/ps-tarifs-21-03.csv -d list(range(10,25))
# Aide : python adressesmatcher.py --help
