Fusion de 2 PS
python ps_merge.py ps1_key ps2_inpp
Cette commande va fusionner ps1 dans ps2, l'inpp final sera celui de ps2_inpp
ps1_key correspond au ps.key du ps qui va être fusionné
ps2_inpp correspond au ps.key du ps qui va recevoir les tarifs de ps1

Backup
A faire 1 fois par mois après les imports PS
Clic droit sur la base de données icip
Backup
Choisir le filename
Cliquer sur dump options
Passer Pre-data, Post-data et Data à YES
Cliquer sur Backup

Restore
Clic droit sur la base de données icip
Restore
Cliquer sur les ... de Filname et choisir le .bak
Cliquer sur Restore Options
Passer Pre-data, Post-data et Data à YES
Passer Clean before restore à YES
Restore
